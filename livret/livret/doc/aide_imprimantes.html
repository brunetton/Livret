Différents types d'imprimantes

Le déroulement des opérations

  1. ouvrir un fichier PDF original
  2. choisir son type d'imprimante
  3. générer le livret, c'est à dire un ou deux fichiers PDF, selon les options choisies

Pourquoi plusieurs types d'imprimantes ?

Le but de ce logiciel est de permettre d'imprimer des livrets de sorte qu'à la fin de l'impression, le livret soit présenté prêt à être plié. Dans le cas d'une imprimante non recto-verso par exemple, l'utilisateur ne doit pas avoir à effectuer une inversion manuelle de l'ordre des feuilles entre l'impression des rectos et celle des versos.

Selon l'imprimante dont on dispose, l'ordre des pages à imprimer n'est pas le même. C'est pour cela qu'il faut signaler au logiciel le type d'imprimante sur laquelle le livret sera imprimé.

Connaître son type d'imprimante

Généralités

Il est important de noter ici que les noms «jet d'encre» et «laser» ne correspondent pas forcément à une imprimante dont la tête d'impression utilise la technologie jet d'encre; mais correspond à une imprimante dont la tête d'impression imprime sur le dessus de la feuille donnée dans le bac d'entrée et présente cette face imprimée vers le haut. Ce n'est pas le cas des grosses imprimantes laser de bureau qui impriment généralement sur le recto de la feuille donnée dans le bac et restituent cette feuille à l'envers. On ne s'occupe en aucun cas ici de la technologie utilisée pour imprimer (tête d'impression jet d'encre ou laser).

Imprimantes non recto-verso

Représentation shématique

Pour dire :

Mon imprimante est une imprimante non recto-verso qui imprime sur le dessous de la page donnée et la ressort à l'envers, on utilise ici la notation schématique :

original -> (recto après impression, verso après impression)

Ce shema indique qu'on a mis dans l'imprimante une feuille sur laquelle on a tracé un trait vertical. On imprime sur cette feuille un document qui contient un trait horizontal. La feuille qui sort de l'imprimante montre à l'utilisateur le trait horizontal et le trait vertical est au dos.

Comment imprimer le ou les fichiers PDF générés ?

Type ``jet d'encre'' ou ``laser''

  1. imprimer le fichier contenant les rectos (typiquement document-rectos.pdf)
  2. prendre le tas de feuilles donné par l'imprimante et le remettre tel quel dans le bac d'impression
  3. imprimer le fichier contenant les versos (typiquement document-versos.pdf)
  4. le livret est imprimé et prêt à être plié en deux pour former un livre

Type ``imprimante à plat'' ou ``imprimante à plat avec tête d'impression sous la feuille''

  1. imprimer le fichier contenant les rectos (typiquement document-rectos.pdf)
  2. prendre le tas de feuilles donné par l'imprimante, le retourner (selon le côté long des feuilles : on inverse les rectos et les versos) et remettre le tas de feuilles retourné dans le bac de l'imprimante.
  3. imprimer le fichier contenant les versos (typiquement document-versos.pdf)
  4. le livret est imprimé et prêt à être plié en deux pour former un livre

imprimantes recto-verso

Pour les imprimantes recto-verso, il suffit d'imprimer le fichier pdf généré. Le livret ressort prêt à être plié pour être lu comme un livre.
